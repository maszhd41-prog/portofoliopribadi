Proyek #01 Web Development

MEMBUAT WEBSITE E-COMMERCE "KARYA REMAJA"

Deskripsi Proyek

Website Karya Remaja merupakan proyek perancangan sistem e-commerce yang dibuat untuk membantu pemasaran dan penjualan produk fashion berbahan kulit secara digital. Produk yang dijual meliputi sepatu, tas, sandal, sabuk, dan berbagai aksesoris fashion lainnya. Proyek ini bertujuan untuk memberikan solusi bagi pelaku usaha agar dapat memperluas jangkauan pasar, meningkatkan efisiensi transaksi, serta memberikan pengalaman berbelanja yang lebih mudah bagi pelanggan.

Dalam proyek ini, dilakukan analisis kebutuhan sistem secara menyeluruh mulai dari kebutuhan pengguna, kebutuhan fungsional, hingga kebutuhan nonfungsional. Sistem dirancang agar pelanggan dapat melakukan registrasi akun, melihat katalog produk, mencari produk berdasarkan kategori, memasukkan barang ke keranjang belanja, melakukan pemesanan, serta memantau status pesanan secara online.

Tahapan Pengembangan diawali dengan identifikasi proses bisnis yang berjalan, kemudian dilanjutkan dengan pembuatan Software Requirement Specification (SRS), perancangan Use Case Diagram, Activity Diagram, dan rancangan database.

Tanggung Jawab

  • 01 Melakukan analisis kebutuhan sistem
  • 02 Menyusun dokumen SRS (Software Requirement Specification)
  • 03 Membuat UML Diagram (Use Case & Activity)
  • 04 Merancang struktur database
  • 05 Mendesain antarmuka pengguna
  • 06 Menyusun alur transaksi dan proses bisnis

Teknologi yang Digunakan

HTML CSS MySQL UML

Proyek #03 Database MySQL

ANALISIS KEBUTUHAN DATA

Deskripsi Proyek

Proyek ini merupakan serangkaian kegiatan praktikum yang berfokus pada perancangan, pengelolaan, dan implementasi basis data menggunakan MySQL. Dalam proyek ini dipelajari bagaimana data dapat disimpan, diorganisasikan, dan dikelola secara efektif untuk mendukung kebutuhan sistem informasi.

Kegiatan dimulai dari analisis kebutuhan data berdasarkan studi kasus tertentu, kemudian dilanjutkan dengan perancangan Entity Relationship Diagram (ERD), normalisasi data, pembuatan tabel, serta penentuan relasi antar tabel. Setelah struktur database selesai, dilakukan implementasi menggunakan SQL untuk mengelola data: penambahan, pengubahan, penghapusan, dan pencarian data.

Selain itu, dilakukan juga pengujian terhadap integritas data dan performa query untuk memastikan database dapat berjalan sesuai kebutuhan sistem. Proyek ini memberikan pemahaman mendalam mengenai pentingnya basis data dalam mendukung operasional sebuah aplikasi atau sistem informasi.

Tanggung Jawab

  • 01 Membuat Entity Relationship Diagram (ERD)
  • 02 Melakukan normalisasi database
  • 03 Membuat tabel dan relasi antar tabel
  • 04 Menulis query SQL (CRUD)
  • 05 Menguji hasil implementasi database
